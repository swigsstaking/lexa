## Bloc B — Route upload + Ingestion

### B.1 — Route

Ajouté dans `apps/backend/src/routes/connectors.ts` :

- `POST /connectors/camt053/upload` — multipart/form-data, champ `file`, auth JWT Bearer (`requireAuth`)
- Multer mémoire, limite 10 MB, filtre `.xml` / `application/xml` / `text/xml`
- Parse XML → `parseCamt053()` → `ingestCamt053Transactions()` → HTTP 201
- Réponse : `{ messageId, accountIban, accountName, currency, transactionsCount, ingested, skipped, failed, streamIds, warnings }`

### B.2 — Pipeline ingestion

Fonction `ingestCamt053Transactions()` :
- **Déduplication** : check `metadata->>'txId'` + `source = 'camt053'` avant insertion
- **Event** : `TransactionIngested(source:"camt053")` avec metadata riche (messageId, statementId, accountIban, counterpartyName, valueDate, structuredRef)
- **Classifier** : fire-and-forget via `enqueueLlmCall()` — pas de `await` bloquant
- **Classification background** : append `TransactionClassified` + insert `ai_decisions` en `.then()` avec gestion d'erreur isolée
