- Créé via API Ollama v0.20.3 (`from` + `system` + `parameters`)

### AuditAgent.ts

`apps/backend/src/agents/audit/AuditAgent.ts`

Re-ranking tier 0 : CO 957-963b +0.3, LTVA 70-71 +0.2, nLPD +0.15, LIFD +0.05

### CitationVerifier.ts

`apps/backend/src/services/CitationVerifier.ts`

- `verifyCitation(citation)` : embed → search Qdrant avec filtre loi → match exact article → match fuzzy
- Gestion lois non-standard (Käfer, etc.) → note "non-standard"
- `verifyCitations(list)` : bulk, max 20 citations

### AuditTrail.ts

`apps/backend/src/services/AuditTrail.ts`

- `buildAuditTrail(tenantId, year)` : events + ai_decisions JOIN → timeline structurée
- Stats : totalEvents, totalAiDecisions, averageConfidence, lowConfidenceCount, eventTypes
- Base légale : CO 958f + LTVA 70

### Routes

- `POST /agents/audit/ask` — agent conversationnel
- `POST /audit/verify-citations` — vérification bulk citations
- `GET /audit/trail/:year` — timeline structurée
