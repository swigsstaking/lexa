## Découvertes importantes

- **Ollama v0.20.3 API breaking change** : `POST /api/create` requiert `from` + `system` + `parameters` séparés (pas un bloc `modelfile` texte)
- **bdlf.fr.ch JSON API** : `GET /api/fr/texts_of_law/{sn}/show_as_json` retourne `text_of_law.selected_version.json_content.document.content` avec nœuds `{type, html_content.fr, text.fr, children[]}`
- **PM2 mode dev** : le backend lexa tourne en `npm run dev` (tsx), il faut déployer les sources `.ts`, pas juste le `dist/`
