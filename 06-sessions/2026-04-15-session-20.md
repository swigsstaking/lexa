## Leçons

- En mode cluster PM2, `pm2 start --env development` met tous les workers sur le port dev (3003), cassant le proxy Nginx (qui attend 3004). Toujours utiliser le mode production ou configurer Nginx avec le bon port.
- `LEXA_WEBHOOK_SECRET` doit être présent dans les `.env` des 2 apps avant tout développement bidirectionnel.
- Le test HMAC avec une signature fixe (TS invariant dans le script) peut provoquer des faux négatifs si la protection replay est trop serrée — utiliser `$(date +%s%3N)` dans chaque itération.
